Ce guide accompagne de la création de compte à la génération d'un premier document PDF. En quelques minutes, tout est opérationnel.

## Démarrage rapide

<Steps>
  <Step title="Créer un compte">
    Se rendre sur [topo.ninja](https://topo.ninja) et cliquer sur **Commencer** ou **Se connecter**. Il est possible de créer un compte avec une adresse email ou de se connecter directement via Google.

    <Tip>
      La création de compte est gratuite et ne nécessite pas de carte bancaire. Un essai Premium de 7 jours est automatiquement activé pour permettre de découvrir toutes les fonctionnalités.
    </Tip>
  </Step>

  <Step title="Choisir un plan">
    Topo.ninja propose plusieurs plans adaptés à différents besoins :

    | Plan | Accès |
    |------|-------|
    | **Standard** | Fiche de contrôle, Fiche par axe, Fiche signalétique, Convertisseur de coordonnées |
    | **Premium** | + Résection 3D, fonctionnalités avancées |
    | **Enterprise** | Gestion d'équipe, comptes partagés |

    Le changement de plan est possible à tout moment depuis l'espace abonnement. Un essai gratuit de 7 jours avec accès Premium est offert à chaque nouvel utilisateur.
  </Step>

  <Step title="Découvrir le tableau de bord">
    Après connexion, le tableau de bord s'affiche. Il présente la liste des outils disponibles, organisés par catégorie :

    - **Implantation** : Fiches de contrôle et fiches par axe
    - **Mesure** : Générateur de fiche signalétique
    - **Conversion** : Convertisseur de coordonnées
    - **Planification** : Résection 3D

    Cliquer sur un outil pour l'ouvrir. Chaque outil est indépendant : il est possible de travailler sur plusieurs projets en parallèle.
  </Step>

  <Step title="Créer un premier projet">
    Prenons l'exemple de la **Fiche de contrôle** :

    1. Ouvrir l'outil **Fiche de contrôle**
    2. Remplir les informations du projet : client, chantier, opérateur, référence
    3. Ajouter les points topographiques (manuellement ou par import CSV)
    4. Des notes et des photos peuvent être ajoutées à chaque point

    <Info>
      Les données sont sauvegardées automatiquement dans le navigateur. Il est possible de fermer l'onglet et de revenir plus tard sans perdre le travail en cours.
    </Info>
  </Step>

  <Step title="Générer un premier PDF">
    Une fois les points saisis et les informations complétées, cliquer sur le bouton **Générer PDF**. Le document est créé directement dans le navigateur et téléchargé automatiquement.

    Le PDF contient :
    - Les informations du projet (client, chantier, date, opérateur)
    - Le tableau complet des points avec leurs coordonnées
    - Un aperçu cartographique (si disponible)
    - Les notes et photos associées aux points

    <Tip>
      Vérifier l'aperçu cartographique avant de générer le PDF. Si les points n'apparaissent pas sur la carte, vérifier que les coordonnées sont dans le bon système de référence.
    </Tip>
  </Step>
</Steps>
