## Résumé

Crée une fiche de contrôle organisée par axes de référence. Chaque axe est défini par deux points. L'outil calcule automatiquement la distance orthogonale et la distance le long de l'axe pour chaque point assigné.

## Principe

L'outil prend en entrée des axes de référence (définis par deux points chacun) et des points topographiques (X, Y, Z). Chaque point est assigné à un axe. L'outil calcule alors deux distances : la projection du point sur l'axe (abscisse curviligne) et l'écart perpendiculaire à l'axe (distance orthogonale, avec signe). Le PDF généré est organisé par axe, avec les distances calculées pour chaque point.

Le calcul des distances orthogonales est détaillé dans [Distances orthogonales](/fr/formules/distances-orthogonales).

## Étapes d'utilisation

**Étape 1 -- Définir les axes de référence** Un axe est défini par deux points (départ et arrivée). Renseigner les coordonnées des extrémités. Plusieurs axes peuvent être définis dans un même projet.

**Étape 2 -- Ajouter les points** Saisir les points manuellement ou les importer par CSV. Chaque point est défini par son nom et ses coordonnées X, Y, Z.

**Étape 3 -- Assigner les points aux axes** Associer chaque point à l'axe par rapport auquel les distances doivent être calculées. Un point ne peut être assigné qu'à un seul axe. Pour contrôler un point par rapport à plusieurs axes, le dupliquer.

**Étape 4 -- Générer le PDF** Le PDF est organisé par axe. Pour chaque axe : définition (coordonnées des deux points), tableau des points assignés avec distances calculées, informations du projet.

## Détails techniques

### Convention de signe

Le sens de l'axe (du point de départ vers le point d'arrivée) détermine le signe de la distance orthogonale :

| Position du point                                 | Distance orthogonale |
| ------------------------------------------------- | -------------------- |
| À droite de l'axe (dans le sens départ → arrivée) | Positive (+)         |
| À gauche de l'axe                                 | Négative (-)         |
| Sur l'axe                                         | Zéro (0)             |

### Colonnes de l'éditeur

| Colonne                   | Type          | Description                                       |
| ------------------------- | ------------- | ------------------------------------------------- |
| Nom                       | Texte         | Identifiant du point                              |
| X, Y, Z                   | Texte         | Coordonnées du point                              |
| Distance le long de l'axe | Lecture seule | Position projetée sur l'axe (abscisse curviligne) |
| Distance orthogonale      | Lecture seule | Écart perpendiculaire à l'axe (avec signe)        |

Les colonnes de distance sont calculées automatiquement à partir des coordonnées du point et de la définition de l'axe.

### Format de définition d'un axe

Chaque axe est défini par un nom et deux points (X1, Y1, Z1) et (X2, Y2, Z2). Le vecteur directeur de l'axe va du point 1 au point 2.
