## Résumé

Crée une fiche de contrôle PDF à partir de points topographiques (X, Y, Z) avec notes et photos optionnelles. Les points sont saisis manuellement ou importés par CSV, puis exportés en PDF formaté.

## Principe

L'outil prend en entrée une liste de points avec leurs coordonnées (X, Y, Z), enrichis éventuellement de notes textuelles et de photos. Il génère un document PDF contenant l'en-tête du projet, le tableau des points, les notes, les photos et un aperçu cartographique.

Les coordonnées doivent être dans un système projeté (Lambert 93, UTM, etc.) pour que l'aperçu cartographique fonctionne. Les points en système local sans géoréférencement ne s'affichent pas sur la carte.

## Étapes d'utilisation

**Étape 1 -- Remplir les informations du projet**
Compléter le formulaire projet : client, chantier, opérateur, référence, date. Ces champs apparaissent dans l'en-tête du PDF. Tous sont optionnels.

**Étape 2 -- Ajouter les points**
Saisir les points manuellement (nom, X, Y, Z) ou importer un fichier CSV. L'import détecte automatiquement les colonnes. Voir [Import/export CSV](/fr/fonctionnalites/import-export-csv) pour les formats acceptés.

**Étape 3 -- Enrichir les points (optionnel)**
Ajouter des notes ou des photos à chaque point. Notes et photos sont incluses dans le PDF à côté des coordonnées du point correspondant. Voir [Photos de points](/fr/fonctionnalites/photos-points) pour les spécifications.

**Étape 4 -- Générer le PDF**
Le PDF est créé dans le navigateur et téléchargé automatiquement. Il contient : en-tête projet, tableau des points avec coordonnées, notes, photos et aperçu cartographique.

## Détails techniques

### Champs du formulaire projet

| Champ | Description |
|-------|-------------|
| Client | Nom du maître d'ouvrage ou donneur d'ordre |
| Chantier | Nom ou adresse du chantier |
| Opérateur | Nom du topographe |
| Référence | Numéro de dossier ou de commande |
| Date | Date du relevé |

### Format CSV attendu

Le format minimal est `Nom,X,Y,Z`. La colonne Z est optionnelle. Les en-têtes reconnus sont détaillés dans [Import/export CSV](/fr/fonctionnalites/import-export-csv).

### Photos

Les photos sont compressées côté client (max 1 Mo, 1024 px côté le plus long) et stockées en Base64 dans le navigateur. Formats acceptés : JPG, PNG, WebP.

### Aperçu cartographique

<Warning>
L'aperçu cartographique nécessite des coordonnées dans un système projeté reconnu (Lambert 93, UTM, etc.). Les coordonnées en système local sans géoréférencement ne s'affichent pas sur la carte.
</Warning>
