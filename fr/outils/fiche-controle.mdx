La **Fiche de contrôle** est l'outil de base pour générer des rapports terrain. Les points topographiques sont saisis avec leurs coordonnées, enrichis de notes ou de photos, puis exportés en PDF formaté, prêt à imprimer ou à transmettre.

<Note>
  **Disponible sur :** Tous les plans
</Note>

## À quoi sert cet outil ?

Lors d'un contrôle d'implantation sur chantier, des points sont relevés et un document récapitulatif doit être produit. La fiche de contrôle permet de :

- Rassembler toutes les informations du projet (client, chantier, opérateur, référence, date)
- Lister les points avec leurs coordonnées X, Y, Z
- Ajouter des observations ou des photos par point
- Visualiser les points sur une carte interactive
- Générer un PDF professionnel en un clic

Ce document se transmet au maître d'ouvrage, au conducteur de travaux, ou se conserve dans le dossier de chantier.

## Comment l'utiliser

<Steps>
  <Step title="Remplir les informations du projet">
    Commencez par compléter le formulaire du projet :
    - **Client** : le nom du maître d'ouvrage ou du donneur d'ordre
    - **Chantier** : le nom ou l'adresse du chantier
    - **Opérateur** : le nom du topographe
    - **Référence** : le numéro de dossier ou de commande
    - **Date** : la date du relevé

    Ces informations apparaîtront dans l'en-tête du PDF.
  </Step>

  <Step title="Ajouter les points">
    Deux options sont disponibles pour ajouter des points :

    **Saisie manuelle** : cliquez sur le bouton d'ajout et renseignez le nom du point ainsi que ses coordonnées X, Y et Z.

    **Import CSV** : importez un fichier CSV contenant les points. Le format attendu est simple : une colonne par champ (nom, X, Y, Z). L'outil détecte automatiquement les colonnes.

    <Tip>
      Pour un grand nombre de points, l'import CSV fait gagner un temps considérable. Les données peuvent être exportées depuis un carnet de terrain ou un logiciel de post-traitement.
    </Tip>
  </Step>

  <Step title="Enrichir les points (optionnel)">
    Pour chaque point, il est possible de :
    - Ajouter une **note** (observation terrain, remarque sur la matérialisation, etc.)
    - Joindre une **photo** (prise avec un téléphone ou une tablette)

    Les notes et photos apparaîtront dans le PDF à côté des coordonnées du point correspondant.
  </Step>

  <Step title="Vérifier sur la carte">
    L'aperçu cartographique affiche tous les points sur un fond de carte. Utilisez-le pour vérifier visuellement que les points sont cohérents et correctement positionnés.

    <Warning>
      Si les points n'apparaissent pas sur la carte, vérifiez que les coordonnées sont dans un système projeté reconnu (Lambert 93, UTM, etc.) et non dans un système local sans géoréférencement.
    </Warning>
  </Step>

  <Step title="Générer le PDF">
    Cliquez sur **Générer PDF**. Le document est créé dans le navigateur et téléchargé automatiquement. Il contient :
    - L'en-tête avec les informations du projet
    - Le tableau des points et coordonnées
    - Les notes et photos
    - Un aperçu cartographique
  </Step>
</Steps>

## Fonctionnalités

### Formulaire projet

Le formulaire projet regroupe les métadonnées du relevé. Tous les champs sont optionnels, mais remplir au minimum le client et le chantier rend le document plus professionnel et plus facile à retrouver.

### Import et export CSV

Les points peuvent être importés depuis un fichier CSV et exportés à tout moment. Cela permet de :
- Reprendre des données issues d'une station totale ou d'un récepteur GNSS
- Sauvegarder le travail pour le reprendre plus tard
- Partager les données avec un collègue

### Photos de points

Chaque point peut recevoir une ou plusieurs photos. Sur le terrain, photographiez la matérialisation du point (clou, piquet, repère) directement depuis un téléphone ou une tablette. La photo sera incluse dans le PDF final.

### Aperçu cartographique

La carte interactive montre l'ensemble des points. Il est possible de zoomer, déplacer la vue et vérifier la cohérence du relevé avant de générer le PDF.
