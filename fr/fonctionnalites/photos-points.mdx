## Résumé

Associe des photos aux points topographiques. Les images sont compressées automatiquement côté client (pas d'envoi serveur) et intégrées dans les PDF générés.

## Principe

Chaque point peut recevoir une ou plusieurs photos. Les images sont compressées dans le navigateur avant d'être stockées en Base64 dans les données du projet. Lors de la génération du PDF, les photos sont automatiquement incluses à côté des données du point correspondant. Aucune image n'est envoyée vers un serveur.

La fonctionnalité est disponible dans : Fiche signalétique, Fiche de contrôle, Fiche axe.

## Étapes d'utilisation

**Étape 1 -- Sélectionner un point**
Dans l'un des outils compatibles, cliquer sur le point auquel ajouter une photo.

**Étape 2 -- Ajouter une photo**
Glisser-déposer l'image sur la zone de dépôt ou utiliser le sélecteur de fichier.

**Étape 3 -- Vérifier la compression**
L'image est compressée automatiquement. Le résultat est visible dans l'interface.

**Étape 4 -- Ajouter d'autres photos (optionnel)**
Répéter l'opération pour ajouter plusieurs photos au même point.

**Étape 5 -- Générer le PDF**
Les photos sont automatiquement intégrées dans le PDF à côté des données du point correspondant.

## Détails techniques

### Spécifications

| Paramètre | Valeur |
|-----------|--------|
| Taille max par image | 1 Mo (après compression) |
| Dimension max | 1024 px (côté le plus long) |
| Formats acceptés | JPG, PNG, WebP |
| Stockage | Local (navigateur), encodé en Base64 |
| Compression | Automatique, côté client |
| Inclusion PDF | Automatique à la génération |

### Outils compatibles

| Outil | Support photos |
|-------|---------------|
| Fiche signalétique | Par point et par paire |
| Fiche de contrôle | Par point |
| Fiche axe | Par point |

### Stockage

Les photos sont stockées dans les données du projet en Base64. Elles sont sauvegardées avec le projet et restaurées à la réouverture. Aucun téléchargement vers un serveur.
