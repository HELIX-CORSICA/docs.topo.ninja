L'editeur en masse vous offre une experience de tableur directement dans Topo.ninja. Base sur la librairie Handsontable, il vous permet de manipuler vos donnees avec la meme fluidite qu'un tableur classique, sans quitter l'application.

<Note>
  Disponible sur les plans **Premium** et **Enterprise**.
</Note>

## Comment ca fonctionne

L'editeur en masse remplace les formulaires traditionnels par une grille de cellules modifiables. Chaque ligne represente un point ou une entree, et chaque colonne correspond a un champ (nom, coordonnees, axe, reference, etc.).

## Utilisation

<Steps>
  <Step title="Ouvrir l'editeur">
    Accedez a l'un des outils compatibles (Fiche signaletique, Fiche axe ou Resection 3D). L'editeur en masse s'affiche automatiquement sous forme de grille.
  </Step>
  <Step title="Saisir ou coller vos donnees">
    Cliquez sur une cellule pour la modifier. Vous pouvez aussi coller des donnees directement depuis Excel ou Google Sheets avec **Ctrl+V**.
  </Step>
  <Step title="Naviguer dans la grille">
    Utilisez **Tab** pour passer a la cellule suivante, **Entree** pour valider et descendre, ou les **fleches directionnelles** pour vous deplacer librement.
  </Step>
  <Step title="Manipuler les lignes">
    Faites un clic droit pour ouvrir le menu contextuel : ajoutez ou supprimez des lignes selon vos besoins.
  </Step>
  <Step title="Utiliser la poignee de remplissage">
    Cliquez sur le coin inferieur droit d'une cellule et faites glisser vers le bas pour copier sa valeur dans les cellules suivantes, exactement comme dans Excel.
  </Step>
</Steps>

## Types de colonnes

L'editeur propose differents types de colonnes selon le contexte :

| Type de colonne | Description | Exemple |
|-----------------|-------------|---------|
| **Case a cocher** | Active ou desactive une ligne | Inclusion dans le calcul |
| **Texte** | Saisie libre | Nom du point, coordonnees |
| **Liste deroulante** | Choix parmi des options | Assignation d'axe, point de reference |
| **Lecture seule** | Valeurs calculees automatiquement | Ecarts, distances |

## Fonctionnalites detaillees

<AccordionGroup>
  <Accordion title="Copier-coller depuis Excel">
    Selectionnez vos cellules dans Excel ou Google Sheets, copiez-les avec **Ctrl+C**, puis collez-les dans l'editeur avec **Ctrl+V**. La structure lignes/colonnes est conservee. C'est la methode la plus rapide pour transferer des donnees existantes.
  </Accordion>
  <Accordion title="Tri et filtrage">
    Cliquez sur l'en-tete d'une colonne pour trier les donnees par ordre croissant ou decroissant. Utilisez les filtres pour afficher uniquement les lignes qui correspondent a vos criteres.
  </Accordion>
  <Accordion title="Redimensionnement">
    Faites glisser les bordures des en-tetes de colonnes ou de lignes pour ajuster leur taille. Pratique quand les noms de points sont longs ou que vous voulez voir plus de donnees a l'ecran.
  </Accordion>
  <Accordion title="Menu contextuel (clic droit)">
    Un clic droit sur la grille ouvre un menu avec les options suivantes :
    - **Inserer une ligne au-dessus / en-dessous**
    - **Supprimer la ligne**
    - **Copier / Coller**
  </Accordion>
</AccordionGroup>

## Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| **Tab** | Cellule suivante |
| **Shift + Tab** | Cellule precedente |
| **Entree** | Valider et descendre |
| **Echap** | Annuler la modification |
| **Ctrl + C** | Copier |
| **Ctrl + V** | Coller |
| **Ctrl + Z** | Annuler |
| **Fleches** | Navigation dans la grille |

## Astuces

<Tip>
  **Coller depuis Excel** : si vous avez deja vos points dans un tableur, selectionnez la plage de cellules, copiez, puis collez directement dans l'editeur. C'est beaucoup plus rapide que la saisie manuelle.
</Tip>

<Tip>
  **Poignee de remplissage** : pour repeter une meme valeur sur plusieurs lignes (par exemple un axe de reference), saisissez la valeur une fois puis faites glisser la poignee vers le bas.
</Tip>

<Tip>
  **Colonnes en lecture seule** : les colonnes grises contiennent des valeurs calculees (ecarts, distances). Elles se mettent a jour automatiquement lorsque vous modifiez les donnees sources.
</Tip>
