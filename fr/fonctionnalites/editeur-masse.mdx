L'éditeur en masse vous offre une expérience de tableur directement dans Topo.ninja. Basé sur la librairie Handsontable, il vous permet de manipuler vos données avec la même fluidité qu'un tableur classique, sans quitter l'application.

<Note>
  Disponible sur les plans **Premium** et **Enterprise**.
</Note>

## Comment ça fonctionne

L'éditeur en masse remplace les formulaires traditionnels par une grille de cellules modifiables. Chaque ligne représente un point ou une entrée, et chaque colonne correspond à un champ (nom, coordonnées, axe, référence, etc.).

Vous retrouvez l'éditeur dans les outils suivants :

## Utilisation

<Steps>
  <Step title="Ouvrir l'éditeur">
    Accédez à l'un des outils compatibles (Fiche signalétique, Fiche axe ou Résection 3D). L'éditeur en masse s'affiche automatiquement sous forme de grille.
  </Step>
  <Step title="Saisir ou coller vos données">
    Cliquez sur une cellule pour la modifier. Vous pouvez aussi coller des données directement depuis Excel ou Google Sheets avec **Ctrl+V**.
  </Step>
  <Step title="Naviguer dans la grille">
    Utilisez **Tab** pour passer à la cellule suivante, **Entrée** pour valider et descendre, ou les **flèches directionnelles** pour vous déplacer librement.
  </Step>
  <Step title="Manipuler les lignes">
    Faites un clic droit pour ouvrir le menu contextuel : ajoutez ou supprimez des lignes selon vos besoins.
  </Step>
  <Step title="Utiliser la poignée de remplissage">
    Cliquez sur le coin inférieur droit d'une cellule et faites glisser vers le bas pour copier sa valeur dans les cellules suivantes, exactement comme dans Excel.
  </Step>
</Steps>

## Types de colonnes

L'éditeur propose différents types de colonnes selon le contexte :

| Type de colonne | Description | Exemple |
|-----------------|-------------|---------|
| **Case à cocher** | Active ou désactive une ligne | Inclusion dans le calcul |
| **Texte** | Saisie libre | Nom du point, coordonnées |
| **Liste déroulante** | Choix parmi des options | Assignation d'axe, point de référence |
| **Lecture seule** | Valeurs calculées automatiquement | Écarts, distances |

## Fonctionnalités détaillées

<AccordionGroup>
  <Accordion title="Copier-coller depuis Excel">
    Sélectionnez vos cellules dans Excel ou Google Sheets, copiez-les avec **Ctrl+C**, puis collez-les dans l'éditeur avec **Ctrl+V**. La structure lignes/colonnes est conservée. C'est la méthode la plus rapide pour transférer des données existantes.
  </Accordion>
  <Accordion title="Tri et filtrage">
    Cliquez sur l'en-tête d'une colonne pour trier les données par ordre croissant ou décroissant. Utilisez les filtres pour afficher uniquement les lignes qui correspondent à vos critères.
  </Accordion>
  <Accordion title="Redimensionnement">
    Faites glisser les bordures des en-têtes de colonnes ou de lignes pour ajuster leur taille. Pratique quand les noms de points sont longs ou que vous voulez voir plus de données à l'écran.
  </Accordion>
  <Accordion title="Menu contextuel (clic droit)">
    Un clic droit sur la grille ouvre un menu avec les options suivantes :
    - **Insérer une ligne au-dessus / en-dessous**
    - **Supprimer la ligne**
    - **Copier / Coller**
  </Accordion>
</AccordionGroup>

## Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| **Tab** | Cellule suivante |
| **Shift + Tab** | Cellule précédente |
| **Entrée** | Valider et descendre |
| **Échap** | Annuler la modification |
| **Ctrl + C** | Copier |
| **Ctrl + V** | Coller |
| **Ctrl + Z** | Annuler |
| **Flèches** | Navigation dans la grille |

## Astuces

<Tip>
  **Coller depuis Excel** : si vous avez déjà vos points dans un tableur, sélectionnez la plage de cellules, copiez, puis collez directement dans l'éditeur. C'est beaucoup plus rapide que la saisie manuelle.
</Tip>

<Tip>
  **Poignée de remplissage** : pour répéter une même valeur sur plusieurs lignes (par exemple un axe de référence), saisissez la valeur une fois puis faites glisser la poignée vers le bas.
</Tip>

<Tip>
  **Colonnes en lecture seule** : les colonnes grises contiennent des valeurs calculées (écarts, distances). Elles se mettent à jour automatiquement lorsque vous modifiez les données sources.
</Tip>
