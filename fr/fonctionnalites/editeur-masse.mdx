## Résumé

Tableur intégré basé sur Handsontable, permettant d'éditer les points en masse. Chaque ligne représente un point, chaque colonne un champ. Supporte le copier-coller depuis Excel/Sheets, la poignée de remplissage, le tri, le filtrage et le menu contextuel.

## Principe

L'éditeur en masse remplace la saisie par formulaire par une grille de cellules modifiables. Les données peuvent être saisies directement, collées depuis un tableur externe (Excel, Google Sheets) ou modifiées cellule par cellule. Les colonnes calculées (distances, écarts) sont en lecture seule et se mettent à jour automatiquement.

L'éditeur est disponible dans : Fiche signalétique, Fiche axe, Résection 3D.

## Étapes d'utilisation

**Étape 1 -- Ouvrir l'outil**
Accéder à l'un des outils compatibles (Fiche signalétique, Fiche axe ou Résection 3D). L'éditeur en masse s'affiche sous forme de grille.

**Étape 2 -- Saisir ou coller des données**
Cliquer sur une cellule pour la modifier. Pour coller depuis Excel ou Google Sheets : sélectionner les cellules dans le tableur externe, copier (Ctrl+C), puis coller dans l'éditeur (Ctrl+V). La structure lignes/colonnes est conservée.

**Étape 3 -- Utiliser la poignée de remplissage**
Cliquer sur le coin inférieur droit d'une cellule et glisser vers le bas pour propager la valeur dans les cellules suivantes.

**Étape 4 -- Manipuler les lignes**
Clic droit pour ouvrir le menu contextuel : insérer une ligne au-dessus/en-dessous, supprimer la ligne, copier, coller.

## Détails techniques

### Types de colonnes

| Type | Description | Exemple |
|------|-------------|---------|
| Case à cocher | Active ou désactive une ligne | Inclusion dans le calcul |
| Texte | Saisie libre | Nom du point, coordonnées |
| Liste déroulante | Choix parmi des options prédéfinies | Assignation d'axe, type de référence |
| Lecture seule | Valeurs calculées automatiquement (colonnes grises) | Écarts, distances orthogonales |

### Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| Tab | Cellule suivante |
| Shift + Tab | Cellule précédente |
| Entrée | Valider et descendre |
| Échap | Annuler la modification |
| Ctrl + C | Copier |
| Ctrl + V | Coller |
| Ctrl + Z | Annuler |
| Flèches | Navigation dans la grille |

### Outils compatibles

| Outil | Colonnes éditables |
|-------|-------------------|
| Fiche signalétique | Nom, X, Y, Z, notes, photos, rôle |
| Fiche axe | Nom, X, Y, Z, assignation d'axe |
| Résection 3D | Nom, X, Y, Z, type de référence |
