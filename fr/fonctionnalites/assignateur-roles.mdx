L'assignateur de rôles vous permet de trier vos points importés en deux catégories : les **points projet** (coordonnées théoriques / de conception) et les **points implantés** (coordonnées mesurées sur le terrain). Cette étape est essentielle pour la génération des fiches signalétiques.

<Note>
  Disponible sur les plans **Standard**, **Premium** et **Enterprise**.
</Note>

## Comment ça fonctionne

Après l'import de vos points, tous arrivent dans un **pool central** (colonne du milieu). Votre travail consiste à les répartir dans les deux colonnes latérales :

<CardGroup cols={3}>
  <Card title="Points Projet" icon="compass-drafting">
    Coordonnées théoriques issues du plan ou du projet. Ce sont les positions **attendues**.
  </Card>
  <Card title="Pool central" icon="layer-group">
    Tous les points non assignés. C'est le point de départ après chaque import.
  </Card>
  <Card title="Points Implantés" icon="location-crosshairs">
    Coordonnées relevées sur le terrain. Ce sont les positions **réelles** mesurées.
  </Card>
</CardGroup>

## Utilisation

<Steps>
  <Step title="Importer vos points">
    Importez vos points via CSV ou saisie manuelle. Tous les points apparaissent dans le pool central.
  </Step>
  <Step title="Utiliser la détection automatique (recommandé)">
    Activez la détection automatique pour identifier les points implantés grâce à leurs marqueurs de nommage. Topo.ninja recherche des motifs comme :
    - `STKD`, `STKED` (stakeout / staked)
    - `IMP` (implanté)
    - Ou tout préfixe/suffixe personnalisé que vous définissez

    Par exemple, un point nommé `BORNE_A_STKD` sera automatiquement identifié comme point implanté.
  </Step>
  <Step title="Compléter manuellement par glisser-déposer">
    Pour les points qui n'ont pas été détectés automatiquement, glissez-les du pool central vers la colonne appropriée.
  </Step>
  <Step title="Vérifier et valider">
    Vérifiez que chaque colonne contient les bons points, puis validez pour passer à l'étape suivante (appairage).
  </Step>
</Steps>

## Méthodes de sélection

Vous disposez de plusieurs méthodes pour sélectionner des points dans chaque colonne :

| Méthode | Geste | Description |
|---------|-------|-------------|
| **Sélection simple** | Clic | Sélectionne un seul point |
| **Multi-sélection** | Ctrl + Clic | Ajoute ou retire des points un par un |
| **Sélection par plage** | Shift + Clic | Sélectionne tous les points entre le premier et le dernier clic |
| **Lasso / Rectangle** | Clic + glisser | Dessinez une zone de sélection sur les points |

## Détection automatique

La détection automatique analyse le nom de chaque point pour identifier les marqueurs d'implantation :

<AccordionGroup>
  <Accordion title="Marqueurs reconnus par défaut">
    Les tokens suivants sont recherchés dans les noms de points (insensible à la casse) :
    - `stkd` ou `stked` (stakeout / staked)
    - `imp` (implanté)

    Exemples : `BORNE_A_STKD`, `IMP_POINT_12`, `PT05_stked`
  </Accordion>
  <Accordion title="Marqueurs personnalisés">
    Vous pouvez définir vos propres préfixes ou suffixes pour correspondre aux conventions de nommage de votre équipe ou de votre logiciel de terrain. Par exemple :
    - Suffixe `_MES` pour "mesure"
    - Préfixe `R_` pour "relevé"
  </Accordion>
  <Accordion title="Comment ça détecte">
    L'algorithme découpe le nom du point en segments (séparateurs : underscore, tiret, espace) et cherche si l'un des segments correspond à un marqueur connu. Si oui, le point est classé comme implanté.
  </Accordion>
</AccordionGroup>

## Tri, filtrage et recherche

<Tabs>
  <Tab title="Tri">
    Cliquez sur l'en-tête d'une colonne (Nom, X, Y, Z) pour trier les points par ordre croissant ou décroissant. Pratique pour regrouper visuellement les points par zone géographique.
  </Tab>
  <Tab title="Filtrage par coordonnées">
    Filtrez les points selon des critères sur les coordonnées :
    - **Égal à** : valeur exacte
    - **Supérieur à** / **Inférieur à** : une borne
    - **Entre** : plage de valeurs (min et max)

    Utile pour isoler les points d'une zone spécifique.
  </Tab>
  <Tab title="Recherche par nom">
    Utilisez la barre de recherche pour trouver rapidement un point par son nom. La recherche est instantanée et filtre les trois colonnes simultanément.
  </Tab>
</Tabs>

## Annuler / Refaire

L'assignateur conserve un historique de **10 niveaux** d'annulation. Utilisez :
- **Ctrl + Z** pour annuler la dernière action
- **Ctrl + Y** pour refaire une action annulée

<Info>
  L'historique couvre toutes les opérations : glisser-déposer, détection automatique, suppressions, etc.
</Info>

## Menu contextuel

Un clic droit sur un ou plusieurs points sélectionnés ouvre un menu contextuel avec des actions rapides : déplacer vers une colonne, supprimer, etc.

## Astuces

<Tip>
  **Conventions de nommage** : adoptez une convention cohérente sur le terrain (par exemple, ajoutez toujours `_IMP` aux points implantés). La détection automatique fonctionnera alors parfaitement à chaque import.
</Tip>

<Tip>
  **Import séparé** : si vos points projet et vos points implantés sont dans deux fichiers CSV distincts, importez-les en deux temps. Le premier import ira dans le pool, assignez-le à "Points Projet", puis importez le second fichier.
</Tip>

<Tip>
  **Sélection rapide** : pour déplacer un grand nombre de points, utilisez la sélection par lasso plutôt que Ctrl+Clic un par un.
</Tip>
