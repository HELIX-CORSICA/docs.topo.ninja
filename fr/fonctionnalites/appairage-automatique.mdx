## Résumé

Associe chaque point implanté à son point projet correspondant par nom (extraction de marqueurs) ou par distance (seuil configurable, 2D ou 3D). Le résultat est un ensemble de paires projet/implanté prêtes pour le calcul des écarts.

## Principe

Après l'assignation des rôles, chaque point implanté (mesuré) doit être associé à son point projet (théorique) pour calculer les écarts. Deux algorithmes complémentaires sont disponibles : l'appairage par nom extrait l'identifiant commun en retirant les marqueurs d'implantation ; l'appairage par distance associe chaque point implanté au point projet le plus proche dans un seuil configurable.

La stratégie recommandée est d'utiliser l'appairage par nom en premier, puis l'appairage par distance pour les points restants.

## Étapes d'utilisation

**Étape 1 -- Vérifier l'assignation des rôles**
S'assurer que les points sont correctement répartis entre "Points Projet" et "Points Implantés". Voir [Assignateur de rôles](/fr/fonctionnalites/assignateur-roles).

**Étape 2 -- Sélectionner le mode d'appairage**
Choisir "Par nom" ou "Par distance".

**Étape 3 -- Configurer les paramètres**
Par nom : vérifier les marqueurs détectés, ajouter des marqueurs personnalisés si nécessaire.
Par distance : définir le seuil en mètres, choisir le mode 2D ou 3D.

**Étape 4 -- Lancer l'appairage**
Les associations sont calculées instantanément.

**Étape 5 -- Vérifier et corriger**
Parcourir les paires trouvées. Les points non appairés sont signalés. Corriger manuellement les associations incorrectes si nécessaire.

<Warning>
Vérifier le résultat de l'appairage automatique. Des noms ambigus ou des points proches peuvent produire des associations incorrectes.
</Warning>

## Détails techniques

### Appairage par nom

L'algorithme décompose le nom du point implanté en segments (séparateurs : underscore, tiret, espace), retire les segments correspondant à des marqueurs connus, puis compare le résultat (insensible à la casse) aux noms des points projet.

| Point implanté | Marqueur retiré | Identifiant extrait | Point projet trouvé |
|---------------|----------------|---------------------|---------------------|
| BORNE_A_STKD | STKD | BORNE_A | BORNE_A |
| IMP_PT12 | IMP | PT12 | PT12 |
| AXE5_stked | stked | AXE5 | AXE5 |

Marqueurs reconnus par défaut : `stkd`, `stked`, `imp` (insensible à la casse). Des marqueurs personnalisés peuvent être ajoutés.

### Appairage par distance

L'algorithme calcule la distance entre chaque point implanté et tous les points projet, puis retient le plus proche si la distance est inférieure au seuil.

| Mode | Formule |
|------|---------|
| Distance 2D | `sqrt((x2-x1)^2 + (y2-y1)^2)` |
| Distance 3D | `sqrt((x2-x1)^2 + (y2-y1)^2 + (z2-z1)^2)` |

Paramètres :
- **Seuil de distance** : distance maximale (en mètres) pour accepter un appairage. Au-delà, le point reste non appairé.
- **Mode 2D / 3D** : la 2D ignore l'altitude, la 3D la prend en compte.

Si plusieurs points projet se trouvent dans le seuil, le plus proche est retenu.

### Stratégie recommandée

1. Lancer l'appairage par nom pour traiter les points avec convention de nommage cohérente.
2. Utiliser l'appairage par distance pour les points restants non appairés.
3. Vérifier et corriger manuellement les cas restants.
