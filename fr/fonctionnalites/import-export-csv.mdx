## Résumé

Importe et exporte des points topographiques au format CSV. À l'import, l'outil détecte automatiquement le délimiteur (virgule, point-virgule, tabulation), les en-têtes de colonnes et le format des coordonnées. À l'export, plusieurs formats sont disponibles.

## Principe

L'import analyse le fichier CSV pour détecter le délimiteur, mapper les colonnes aux champs attendus (nom, X, Y, Z) et convertir les formats de coordonnées si nécessaire (DMS vers degrés décimaux). Le fichier doit contenir une ligne d'en-tête. L'export produit un fichier CSV avec les données du projet dans le format choisi.

## Étapes d'utilisation

**Étape 1 -- Préparer le fichier CSV**
Le fichier doit contenir une ligne d'en-tête. Format minimal : `Nom,X,Y,Z`. La colonne Z est optionnelle.

**Étape 2 -- Importer le fichier**
Glisser-déposer le fichier sur la zone d'import ou cliquer sur le bouton d'import pour sélectionner le fichier.

**Étape 3 -- Vérifier le mapping des colonnes**
L'outil détecte automatiquement les colonnes grâce aux en-têtes. Vérifier que chaque colonne est correctement associée (Nom, X, Y, Z) et ajuster si nécessaire.

**Étape 4 -- Valider**
Confirmer l'import. Les points apparaissent dans l'outil en cours d'utilisation.

## Détails techniques

### En-têtes reconnus

| Champ | En-têtes acceptés |
|-------|-------------------|
| Nom | Nom, Name, Point, ID |
| X | X, Est, Easting, E |
| Y | Y, Nord, Northing, N |
| Z | Z, Alt, Altitude, Elevation, H |
| Latitude | Latitude, Lat |
| Longitude | Longitude, Lon, Long |

### Détection du délimiteur

Le délimiteur est détecté automatiquement parmi : virgule (`,`), point-virgule (`;`), tabulation.

<Warning>
En France, le séparateur décimal est la virgule (`1234,56`), ce qui impose le point-virgule comme délimiteur de colonnes. Si les coordonnées semblent incorrectes après import, vérifier la détection du délimiteur.
</Warning>

### Coordonnées DMS

Les coordonnées en degrés-minutes-secondes (ex : `48°51'24.5"N`) sont reconnues et converties automatiquement en degrés décimaux.

### Encodage

Le fichier doit être encodé en UTF-8. Depuis Excel, utiliser "Enregistrer sous" et choisir "CSV UTF-8 (délimité par des virgules)".

### Formats d'export

| Type d'export | Contenu |
|---------------|---------|
| CSV complet | Toutes les colonnes du projet |
| CSV converti | Coordonnées dans le système cible (convertisseur) |
| CSV fusionné | Données sources + résultats dans un seul fichier |
